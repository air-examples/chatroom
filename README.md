# Chatroom

A real-time web chatroom built using [Air](https://github.com/aofei/air).

## Installation

```bash
$ go get github.com/air-examples/chatroom
```

## Usage

```bash
$ cd $GOPATH/src/github.com/air-examples/chatroom
$ go run main.go
```
